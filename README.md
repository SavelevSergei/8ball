Волшебный шар
-

Это консольное RUBY приложение "Волшебный шар",
после его запуска вы можете задать любой вопрос.
Через 3 секунды компьютер предоставит вам ответ.

#### Запуск осуществляется в консоле командой ####

```
ruby 8_ball.rb
```
